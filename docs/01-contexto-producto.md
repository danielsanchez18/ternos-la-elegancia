# 01 — Contexto de producto

## Negocio
Ternos La Elegancia E.I.R.L. vende prendas de vestir formales y presta servicios asociados a sastrería.

## Módulos del sistema
1. Venta directa
2. Confección personalizada
3. Alquiler
4. Arreglos
5. Citas para toma de medidas y servicios
6. Catálogo web integrado
7. Clientes, medidas e historial
8. Pagos y comprobantes
9. Notificaciones
10. Reportes

## Qué se vende
- terno
- saco
- pantalón
- camisa
- blusa
- chaleco
- accesorios
- bundles

## Qué se alquila
- ternos completos
- sacos
- pantalones
- chalecos
- smokings

## Qué no se debe asumir
- no hay múltiples sedes
- no hay múltiples roles internos
- no todos los productos siguen el mismo flujo
- no todas las personalizaciones crean SKU

## Decisiones clave del negocio
- El cliente puede registrarse por su cuenta.
- El admin gestiona catálogo, horarios, atributos, telas, arreglos y estados.
- El catálogo web integra venta, alquiler, arreglos y reservas, pero operativamente las órdenes son separadas.
- La personalización de confección debe ser flexible y escalable.
