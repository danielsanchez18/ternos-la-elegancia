# 05 — Flujos críticos

## Flujo: compra de terno con toma de medidas
1. Cliente navega catálogo.
2. Selecciona terno o saco.
3. El sistema verifica si tiene medidas vigentes.
4. Si no las tiene, exige reserva de cita.
5. El cliente confirma reserva.
6. Se registra adelanto si aplica.
7. Luego del proceso de medición, el admin puede continuar la confección.

## Flujo: confección personalizada
1. Iniciar desde producto base o desde cero.
2. Crear orden de confección.
3. Agregar piezas incluidas.
4. Asociar medidas vigentes o crear nuevas.
5. Elegir tela.
6. Registrar personalización flexible por pieza.
7. Registrar adelanto del 50%.
8. Gestionar pruebas y ajustes.
9. Marcar listo y entregado.

## Flujo: alquiler
1. Seleccionar unidad física disponible.
2. Crear orden de alquiler inmediata.
3. Elegir precio según `estreno` o `normal`.
4. Bloquear unidad hasta devolución.
5. Registrar devolución real.
6. Registrar daño o retraso si hubo.
7. Evaluar si pasa de estreno a normal.

## Flujo: arreglos
1. Registrar prenda o descripción del trabajo.
2. Elegir o crear tipo de arreglo.
3. Definir precio.
4. Registrar cita si aplica.
5. Cambiar estados hasta entrega.

## Flujo: pagos
1. Crear orden.
2. Registrar uno o varios pagos.
3. Permitir mezcla de métodos.
4. Mantener saldo pendiente.
5. Adjuntar voucher si corresponde.
6. Emitir comprobante.
